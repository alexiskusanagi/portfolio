Integrative Project – Administrative Web System
This project was developed as part of the Technical Course in Application Development (Senac, 2025). It is a complete web system with real functionalities for registration, authentication, service management, and scheduling, designed for administrative and institutional environments.

Features
Registration and listing of clients, employees, and services

Employee login and logout with session control

Service scheduling and modification through the backoffice

Sales management and service status tracking

Navigation through specific areas (client, backoffice)

Interface built with HTML, CSS, icons, and custom images

Database integration using PHP and SQL

Technologies Used
PHP for server-side logic

HTML/CSS for page structure and styling

JavaScript (light usage) for basic interactions

SQL for data persistence

XAMPP as the local development environment

Git and GitHub for version control and collaboration

Project Structure
Código
portfolio/
├── areacliente/              # Client-facing interface
├── css/, icons/, img/        # Styles and visual assets
├── utils/, zold/, ode/       # Auxiliary folders
├── login_funcionario.php     # Employee authentication
├── cliente_lista.php         # Client listing
├── servico_cadastra.php      # Service registration
├── ver_agenda.php            # Schedule visualization
├── vendas.php                # Sales management
└── sobre.php, index.php      # Institutional pages
How to Run
Clone the repository:

bash
git clone https://github.com/alexiskusanagi/portfolio.git
Install XAMPP and move the portfolio folder to the htdocs directory. Import the database (available .sql file, if provided).

Access via browser:

Código
http://localhost/portfolio/index.php

Projeto Integrador – Sistema Web Administrativo
Este projeto foi desenvolvido como parte do curso técnico de Desenvolvimento de Aplicativos (Senac, 2025). Trata-se de um sistema web completo, com funcionalidades reais de cadastro, autenticação, controle de serviços e agendamentos, voltado para ambientes administrativos e institucionais.

Funcionalidades
Cadastro e listagem de clientes, funcionários e serviços

Login e logout de funcionários com controle de sessão

Agendamento e alteração de serviços via backoffice

Controle de vendas e status de atendimento

Navegação por áreas específicas (cliente, backoffice)

Interface com HTML, CSS, ícones e imagens customizadas

Integração com banco de dados via PHP e SQL

Tecnologias utilizadas
PHP para lógica de servidor

HTML/CSS para estrutura e estilo das páginas

JavaScript (uso leve) para interações básicas

XAMPP como ambiente de desenvolvimento local

Git e GitHub para versionamento e colaboração
